What if

``` c++
for (int i=0; i<100; i++) 
{
	doSomething();
}

// and

for (int i=0; i<100; i+=2)
{
	doSomethingElse();
}
```

Was

``` c++
doSomething() for 1..99;
while (i=1,2..99) { doSomethingElse(); };
```

Would the world be a better place?
